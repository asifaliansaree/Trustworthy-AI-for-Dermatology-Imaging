# CIFAR-10 Notes

## What is CIFAR-10?
A dataset of 60,000 color images belonging to 10 classes.

## What is a CNN?
A Convolutional Neural Network used for image classification.

## What is a DataLoader?
Loads and batches data efficiently during training.

## What is a Loss Function?
Measures the error between predictions and true labels.

## What is an Optimizer?
Updates model weights to reduce the loss.
